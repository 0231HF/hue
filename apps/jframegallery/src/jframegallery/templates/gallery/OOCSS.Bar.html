Vertically align the buttons

To vertically align the buttons in a ButtonBar simply add a valign attribute.

e.g. <p class=ButtonBar valign=bottom> or <p class=ButtonBar valign=top>

Font Size

The font-size for the contents of a ButtonBar is hard-coded at 11px.

Custom CSS

To override this setting, add the following to your app's custom css file:

.MyApp .ButtonBar > * {
	font-size:22px;
}

Note: replace MyApp with the class of your actual app.

Subclass ButtonBar

Alternatively you can subclass ButtonBar itself.

CSS: Create your subclass.

.ButtonBar.MyCustomButtonBar3 > * {
	font-size:22px;
}

HTML: Add your new class to a ButtonBar element.
